# 八、经验编号建议

建议统一编号：
- `R` = 正式规则（Rule）
- `T` = 试运行经验（Trial）
- `O` = 观察经验（Observe）
- `D` = 废弃经验（Deprecated）

示例：
- `R003` = 第 3 条正式规则
- `T005` = 第 5 条试运行经验
