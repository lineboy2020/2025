# 多因子量化策略开发方案

## 📋 策略概述

**策略名称**: 情绪-资金-形态共振策略  
**策略类型**: 多因子选股 + 趋势跟踪  
**交易频率**: 日线级别，每日盘后选股，次日开盘买入  
**目标收益**: 年化30-50%，最大回撤<20%

---

## 🎯 核心因子体系

### 1️⃣ 市场情绪周期 (权重: 25%)

**数据来源**: `market-emotion` 技能输出

| 情绪周期 | 仓位建议 | 选股方向 | 风险等级 |
|----------|----------|----------|----------|
| 🥶 冰点期 | 10-20% | 超跌反弹、首板 | 高 |
| 🌱 启动期 | 30-50% | 1进2、低位补涨 | 中 |
| 🔥 发酵期 | 50-70% | 龙头、主升浪 | 中低 |
| 🚀 高潮期 | 70-100% | 核心龙头 | 高 |
| 📉 退潮期 | <10% | 空仓或避险 | 极高 |

**具体因子**:
- 情绪周期标签 (冰点/启动/发酵/高潮/退潮)
- 情绪分数 (0-100)
- 涨停家数变化趋势
- 跌停家数变化趋势
- 涨跌比

---

### 2️⃣ 板块轮动 (权重: 25%)

**数据来源**: 
- `ths-smart-stock-picking` 板块热度
- `daily-quant-review` 板块分析
- `kline_eod.duckdb` 板块资金流向

**轮动逻辑**:
```
1. 识别当前主线板块 (涨停数>10的板块)
2. 计算板块强度 = 板块涨停数 / 总涨停数
3. 计算板块资金净流入 (主力净流入占比)
4. 识别板块轮动信号:
   - 老主线退潮 (涨停数连续2日下降)
   - 新主线启动 (新板块涨停数激增)
```

**具体因子**:
- 板块涨停占比
- 板块资金净流入排名
- 板块5日涨幅
- 板块轮动速度

---

### 3️⃣ 主力资金流向 (权重: 25%)

**数据来源**: `kline_eod.duckdb.capital_flow`

**资金指标**:
| 指标 | 计算方法 | 阈值 |
|------|----------|------|
| 主力净流入 | 当日主力买入 - 主力卖出 | >1000万 |
| 主力净流入占比 | 主力净流入 / 成交额 | >10% |
| 连续净流入天数 | 近5日主力净流入为正的天数 | >=3天 |
| 大单占比 | 大单成交额 / 总成交额 | >30% |

**资金流向强度评分**:
```python
def capital_score(row):
    score = 0
    if row['main_net_inflow'] > 10_000_000:  # 主力净流入>1000万
        score += 30
    if row['main_inflow_ratio'] > 0.1:  # 占比>10%
        score += 25
    if row['consecutive_inflow_days'] >= 3:
        score += 25
    if row['large_order_ratio'] > 0.3:
        score += 20
    return min(100, score)
```

### 4️⃣ K线形态多头反转 (权重: 25%)

**技术形态库**:

#### A. 底部反转形态
| 形态 | 特征 | 买入信号 |
|------|------|----------|
| 早晨之星 | 阴线+十字星+阳线，第三日突破第一日高点 | 确认后买入 |
| 锤头线 | 长下影线，实体小，位于下跌趋势底部 | 次日高开买入 |
| 吞没形态 | 阳线完全吞没前一日阴线 | 确认后买入 |
| W底 | 两次探底，第二次成交量萎缩 | 突破颈线买入 |

#### B. 趋势延续形态
| 形态 | 特征 | 买入信号 |
|------|------|----------|
| 多头排列 | 5日>10日>20日>60日均线 | 回踩5日线买入 |
| 突破形态 | 放量突破前期高点/平台 | 突破确认买入 |
| 旗形整理 | 快速上涨后缩量整理 | 突破旗形上轨买入 |

#### C. 量价关系
| 信号 | 特征 | 意义 |
|------|------|------|
| 量价齐升 | 价格上涨+成交量放大 | 上涨趋势确认 |
| 缩量回调 | 价格回调+成交量萎缩 | 洗盘信号 |
| 放量滞涨 | 价格横盘+成交量放大 | 警惕出货 |

## 📊 选股流程

### Step 1: 情绪周期判断
```python
emotion_data = get_emotion_cycle()
if emotion_data['emotion_cycle'] == '退潮期':
    return []  # 空仓
position_limit = emotion_data['position_advice']  # 仓位上限
```

### Step 2: 板块筛选
```python
hot_sectors = get_hot_sectors(min_limit_up=5)
sector_scores = {}
for sector in hot_sectors:
    score = calculate_sector_score(sector)
    sector_scores[sector] = score
top_sectors = sorted(sector_scores.items(), key=lambda x: x[1], reverse=True)[:3]
```

### Step 3: 个股筛选 (在热门板块内)
```python
candidates = []
for sector, _ in top_sectors:
    stocks = get_stocks_in_sector(sector)
    for stock in stocks:
        # 资金流向评分
        capital_score = calculate_capital_score(stock)
        # 技术形态评分
        pattern_score = calculate_pattern_score(stock)
        # 综合评分
        total_score = (
            emotion_data['emotion_score'] * 0.25 +
            sector_scores[sector] * 0.25 +
            capital_score * 0.25 +
            pattern_score * 0.25
        )
        if total_score > 60:
            candidates.append({
                'stock_code': stock,
                'sector': sector,
                'total_score': total_score,
                'emotion_score': emotion_data['emotion_score'],
                'sector_score': sector_scores[sector],
                'capital_score': capital_score,
                'pattern_score': pattern_score
            })
```

### Step 4: 排序精选
```python
# 按综合评分排序
candidates.sort(key=lambda x: x['total_score'], reverse=True)
# 精选Top N
selected = candidates[:15]
```

### Step 5: 仓位分配
```python
# 根据评分分配仓位
for stock in selected:
    weight = stock['total_score'] / sum([s['total_score'] for s in selected])
    stock['position'] = min(weight * position_limit, 0.2)  # 单票最大20%
```

## 🔄 交易规则

### 买入规则
1. **时间**: 次日开盘 (09:30) 或 盘中回踩5日线
2. **价格**: 开盘价或限价单 (不超过前收盘+5%)
3. **数量**: 根据仓位分配计算
4. **条件**: 
   - 大盘不处于退潮期
   - 个股不低开超过-3%
   - 板块仍保持热度

### 卖出规则
1. **止盈**: 浮盈>15%，减仓50%；浮盈>25%，清仓
2. **止损**: 浮亏>-7%，清仓
3. **趋势反转**: 跌破10日线或出现顶部形态，清仓
4. **情绪退潮**: 市场进入退潮期，减仓至10%以下

### 仓位管理
| 情绪周期 | 总仓位 | 单票最大仓位 |
|----------|--------|--------------|
| 冰点期 | 10-20% | 5% |
| 启动期 | 30-50% | 10% |
| 发酵期 | 50-70% | 15% |
| 高潮期 | 70-100% | 20% |
| 退潮期 | <10% | 5% |

## 💡 关键问题讨论

### Q1: 如何处理情绪周期与个股选择的冲突？
**方案**: 情绪周期决定总仓位和选股方向，个股选择在该框架下进行。
- 启动期: 重点选1进2、低位启动股
- 发酵期: 重点选龙头、主升浪股
- 高潮期: 只选核心龙头

### Q2: 板块轮动如何判断？
**方案**: 多维度判断
- 涨停数占比 (>20%为热点)
- 资金净流入排名 (Top 5)
- 5日涨幅 (>10%为强势)
- 持续性 (连续3日保持热度)

### Q3: 如何识别假突破？
**方案**: 多重验证
- 成交量验证 (突破时放量)
- 资金流向验证 (主力资金流入)
- 板块验证 (同板块有跟风股)
- 次日确认 (不低开)

### Q4: 止损止盈如何设置？
**方案**: 动态调整
- 固定止损: -7%
- 移动止盈: 浮盈>15%后，回撤5%止盈
- 时间止损: 持有5日无表现，减仓
