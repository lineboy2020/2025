# 同花顺策略模板项目

## 项目目标
建立一个可复用、文件少而精的量化策略模板项目，覆盖：
- 数据源接口定义
- 历史回测
- 实时计算
- 盘中预警
- 复盘优化

并满足：
- 默认盘中禁止 SDK
- DuckDB 作为主仓
- 少量策略私有数据可使用 Parquet
- 模板先做轻量统计分析与轻量调参
- 带一个示例策略

7. `runner_live.py`
8. `notifier.py`

## 已确认设计决策
- 策略类型：通用双模版（兼容尾盘 + 日内）
- 存储：DuckDB 主仓；少量私有策略数据可用 Parquet
- 盘中：默认禁止 SDK，优先 HTTP
- 通知：控制台 / Markdown / QQ / 钉钉
- 调参：先轻量版，预留扩展位
- 输出：骨架 + 一个示例策略
