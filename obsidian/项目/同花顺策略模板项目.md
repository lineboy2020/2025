# 同花顺策略模板项目

## 项目目标
建立一个可复用、文件少而精的量化策略模板项目，覆盖：
- 数据源接口定义
- 历史回测
- 实时计算
- 盘中预警
- 复盘优化

并满足：
- 默认盘中禁止 SDK
- DuckDB 作为主仓
- 少量策略私有数据可使用 Parquet
- 模板先做轻量统计分析与轻量调参
- 带一个示例策略

## 当前项目路径
- `/root/.openclaw/workspace/projects/ths-strategy-template`

## 文件规划（8个）
1. `README.md`
2. `config.json`

4. `strategy_logic.py`
5. `analyzer.py`
6. `runner_backtest.py`
7. `runner_live.py`
8. `notifier.py`

## 已确认设计决策
- 策略类型：通用双模版（兼容尾盘 + 日内）
- 存储：DuckDB 主仓；少量私有策略数据可用 Parquet
- 盘中：默认禁止 SDK，优先 HTTP
- 通知：控制台 / Markdown / QQ / 钉钉
- 调参：先轻量版，预留扩展位
- 输出：骨架 + 一个示例策略

## 当前版本结论（V1）
- 已完成“第一版可运行模板”落地，可作为后续策略孵化底座。
- 当前模板已经具备：
  - 独立项目目录
  - 唯一配置入口
  - DuckDB 历史数据读取
  - THS HTTP 实时 / 高频 / snapshot 读取
  - 候选池读取与最近非空候选池解析
  - 单文件策略逻辑
  - 回测入口
  - 实时入口
  - 多股票批量运行
  - Markdown 报告输出
- 当前版本有意保留为“轻量可扩展模板”，暂不继续膨胀文件数量。
