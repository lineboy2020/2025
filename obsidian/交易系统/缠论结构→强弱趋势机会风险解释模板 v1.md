# 缠论结构→强弱趋势机会风险解释模板 v1

## 一、目标

本模板用于把现有缠论算法输出：
- 分型
- 笔
- 中枢
- 买卖点
- 卖卖点
- 当前状态机

翻译成用户真正关心的五类结论：

1. 强弱
2. 趋势
3. 机会
4. 风险
5. 涨跌幅度预期

重点不是预测神准，而是：

## 让结构结果变成稳定、可解释、可复盘的判断语言。

---

## 二、输入字段（第一版默认）

### 结构层输入
- `bi`
- `zhongshu`
- `buy_points`
- `sell_points`
- `current_state`

### 可派生输入
- 最后一笔方向
- 最近 5 笔高低点关系
- 当前价格相对最新中枢位置
- 最近是否出现 1B / 2B / 3B
- 最近是否出现 1S / 2S / 3S
- 最近是否存在背驰信号

---

## 三、强弱解释规则 v1

### A. 强
满足以下多数特征：
- 最后一笔为上升笔
- 最近 5 笔低点抬高
- 当前价格或最后一笔低点位于最新中枢上方
- 当前状态接近 `3B` 或 `2B confirmed`
- 最近无明显 `1S / 2S / 3S` 风险信号
- 无明显上涨背驰

解释输出：
- 结构偏强
- 趋势延续概率较高
- 更适合顺势思路，而不是猜顶

### B. 中强
满足部分强势条件，但不完全：
- 最后一笔可能仍向上，或刚进入回调
- 最近低点大体抬高，但回调力度略大
- 价格可能处于中枢上沿附近或刚回到中枢内部
- 可能存在 `2B`，但 `3B` 尚未完全确认

解释输出：
- 结构仍偏多，但趋势强度有所减弱
- 适合等回踩确认或等再度转强

### C. 中性
典型特征：
- 笔来回交替
- 高低点无明显抬高或下移
- 价格在中枢内部震荡
- 当前状态无明确买卖点

解释输出：
- 当前属中枢震荡
- 没有明显优势方向
- 观察胜于出手

### D. 中弱
典型特征：
- 最后一笔转下
- 最近高点不过前高
- 价格回到中枢内部或中枢下沿附近
- 存在 `1S` 或卖点雏形，但未完全走坏

解释输出：
- 结构有转弱迹象
- 上攻效率下降
- 若反弹不能重新站回关键位置，风险会继续增大

### E. 弱
典型特征：
- 最后一笔为下降笔
- 最近低点持续下移
- 跌破中枢下沿
- 出现 `1S / 2S / 3S` 或明显上涨背驰后转弱
- 无有效买点承接

解释输出：
- 当前偏弱
- 反弹更倾向于卖点而不是买点
- 结构风险大于机会

## 四、趋势解释规则 v1

### 1. 上涨趋势
满足以下大部分特征：
- 最近 5 笔高点抬高、低点抬高
- 最新中枢位置相对更高
- 当前处于中枢上方
- 回调不破关键低点或中枢上沿

输出：
- 上涨趋势
- 若当前回调，属于趋势中的正常回踩

### 2. 中枢震荡
满足以下特征：
- 笔交替明显
- 高低点横向波动
- 价格主要运行在中枢内部
- 无明确突破 / 跌破动作

输出：
- 中枢震荡
- 方向未确认
- 等待离开中枢再判断

### 3. 下跌趋势
满足以下大部分特征：
- 最近低点下移、高点下移
- 跌破中枢后反抽不过中枢
- 最后一笔仍为下降笔
- 卖点信号占优

输出：
- 下跌趋势
- 当前反抽更偏减仓或规避风险

### 4. 弱转强修复
满足以下特征：
- 前期明显下跌或偏弱
- 出现 1B 或 2B
- 当前最后一笔开始转上
- 尚未完全进入强趋势，但结构在改善

输出：
- 弱转强修复
- 可重点观察后续是否形成 2买确认 / 3买跟随

### 5. 强转弱分歧
满足以下特征：
- 前期偏强
- 当前出现上涨背驰或 1S
- 价格开始跌回中枢或最后一笔转下

输出：
- 强转弱分歧
- 若后续不能重新转强，趋势可能进入衰退

## 五、机会解释规则 v1

### 1. 1买试错机会
条件：
- 出现 1B
- 下跌创新低但背驰
- 当前仍处在反转初期

输出：
- 左侧试错机会
- 盈亏比高，但确认度偏低

### 2. 2买确认机会
条件：
- 出现 2B
- 回调不破 1买低点
- 结构出现确认

输出：
- 确认型低风险机会
- 比 1买更适合多数情况

### 3. 3买跟随机会
条件：
- 出现 3B
- 价格脱离中枢后回踩不回中枢

输出：
- 趋势跟随机会
- 属于强势股最有价值的结构之一

### 4. 中枢突破待回踩
条件：
- 刚突破中枢
- 但尚未形成标准 3买

输出：
- 可关注但不宜急追
- 等回踩确认会更优

### 5. 无明显机会
条件：
- 当前无 1B / 2B / 3B
- 同时也不处在趋势回踩良好位置

输出：
- 当前无明显结构性机会
- 观望优先

## 六、风险解释规则 v1

### 风险低
特征：
- 处于上升趋势
- 回踩未破关键位置
- 无明显卖点或背驰

输出：
- 当前风险较低
- 主要风险来自短线波动，而非结构破坏

### 风险中
特征：
- 回到中枢内部
- 最后一笔转下
- 有轻微卖点雏形或上涨动能减弱

输出：
- 风险开始抬升
- 需观察是否重新转强或跌破关键位

### 风险高
特征：
- 跌破中枢下沿
- 明确出现 `1S / 2S / 3S`
- 上涨背驰后转弱
- 关键买点低点被跌破

输出：
- 结构风险高
- 现阶段机会小于风险

---

## 七、涨跌幅度预期规则 v1

> 第一版不做神准预测，只做“结构目标区间预期”。

### 向上预期
优先级：
1. 前高
2. 最新中枢高度投射
3. 最近上升笔幅度投射

输出方式：
- 第一目标位：前高附近
- 第二目标位：中枢高度投射区
- 若继续强势，再看趋势延伸

### 向下风险区
优先级：
1. 最新中枢上沿
2. 最新中枢下沿
3. 关键买点低点
4. 最后确认笔起点

输出方式：
- 第一失守位：中枢上沿 / 最近支撑位
- 结构失效位：中枢下沿 / 关键买点低点

## 九、第一版使用原则

1. 先解释结构，再给结论
2. 先判断趋势，再讨论机会
3. 先给区间预期，不给绝对预言
4. 风险与机会必须同时给出
5. 输出必须能让人直接用于复盘和看盘
