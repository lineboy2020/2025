# 🎭 市场情绪周期技能 v3.0 (Market Emotion Cycle)

## 概述

基于**XGBoost模型**和**92情绪周期理论**的A股市场情绪周期识别系统，每日收盘后自动判断当前市场所处情绪周期，并给出**明确的仓位建议和操作思路**。

## 核心功能

### 1. 每日情绪周期判断

**命令**：
```bash
python scripts/main.py --today
```

## 📊 情绪周期定义

| 周期 | 特征描述 | 仓位建议 | 核心策略 |
|:----:|:---------|:--------:|:---------|
| **冰点期** 🥶 | 涨停<30只，跌停多(>50只)，炸板率高，市场恐慌 | **空仓或10%** | 低吸首板，关注超跌新题材 |
| **启动期** 🌱 | 涨停30-50只，跌停减少，情绪回暖，赚钱效应改善 | **30%~50%** | 1进2半路打板，高低切换 |
| **发酵期** 🔥 | 涨停60-90只，连板活跃，龙头主升，板块扩散 | **50%~70%** | 持有龙头，加仓主升浪 |
| **高潮期** 🚀 | 涨停>100只，市场亢奋，一字板多，妖股频现 | **70%~100%** | 重仓龙头，做妖股博弈 |
| **退潮期** 📉 | 涨停减少，跌停增加，分歧加大，情绪降温 | **<10%或空仓** | 空仓避险，等待新周期 |

## 🎯 操作建议详解

### 冰点期 🥶
```
【仓位】空仓或10%以下试探仓位
【策略】低吸首板，关注低位新题材首板、历史低位股
【选股】
  - 低位首板：流通市值<50亿，前期跌幅>30%
  - 新题材首板：政策驱动、事件催化
  - 超跌反弹：连续3日跌幅>15%
【止损】首板次日低于成本价5%止损
【关注信号】
  - 指数企稳放量
  - 龙头股止跌
  - 跌停数明显减少
```

### 启动期 🌱
```
【仓位】30%~50%
【策略】1进2半路打板，高低切换、补涨
【选股】
  - 1进2：前日首板，今日竞价高开3%+
  - 高低切换：龙头断板后的低位补涨股
  - 板块轮动：资金流出板块的龙头首阴
【止损】2板失败次日弱转强失败止损
【关注信号】
  - 涨停数回升
  - 跌停数减少
  - 赚钱效应改善
```

### 发酵期 🔥
```
【仓位】50%~70%
【策略】持有龙头，加仓主升、接龙头为主
【选股】
  - 龙头股：连板最高，加仓主升浪
  - 龙头跟随股：同板块2-3板股
  - 龙头首阴：龙头断板首日低吸
【止损】跌破5日线减仓，破10日线清仓
【关注信号】
  - 龙头股加速（一字板或T字板）
  - 板块持续扩散
  - 成交额放大
```

### 高潮期 🚀
```
【仓位】70%~100%（根据龙头强度）
【策略】重仓龙头，全仓龙头、做妖股
【选股】
  - 核心龙头：板块空间最高股
  - 妖股博弈：超预期连板股
  - 龙头加速：一字板加速阶段跟随
【止损】炸板不封回考虑止盈，次日弱势离场
【关注信号】
  - 一字板增多（警惕）
  - 成交量暴增
  - 散户入场情绪高涨（警惕）
```

### 退潮期 📉
```
【仓位】10%以下或空仓
【策略】空仓避险，等待新周期、切换低位
【选股】
  - 低位新题材：与前期主线无关的新方向
  - 超跌反抽：大跌后技术性反弹（快进快出）
  - 避险板块：防御性板块龙头
【止损】严格执行，不抱幻想，保住本金
【关注信号】
  - 跌停数峰值出现
  - 龙头股补跌完成
  - 指数止跌信号
```

### 独立部署步骤

本技能已完全独立封装，可直接部署到其他平台。部署步骤：

1. **复制技能目录**
```bash
cp -r market-emotion/ /path/to/target/skills/
```

2. **复制必要数据文件**
```bash
# 复制特征数据
cp data/index/emotion_features.parquet market-emotion/data/index/

# 复制模型文件
cp models/emotion_cycle_xgb.pkl market-emotion/models/
```

3. **安装依赖**
```bash
cd market-emotion
pip install -r requirements.txt
```

4. **运行预测**
```bash
python scripts/main.py --today
```

| 源文件 | 目标位置 | 说明 |
|--------|----------|------|

## ⚠️ 注意事项

1. **数据依赖**：
   - 技能直接调用主项目数据库（`data/db/`），无需在技能内维护重复数据
   - 需要先运行主更新链路 `skills/eod-full-update/scripts/run_eod_with_kline_guard.py`，确保主数据库为最新
   - 确保主项目的 `qingxu.parquet`、`zhishu.parquet`、`limit_up.duckdb` 为最新
2. **模型依赖**：需要已训练好的XGBoost模型 (`models/emotion_cycle_xgb_v3.pkl`（默认）)
3. **交易时间**：建议在收盘后（15:30之后）运行，确保数据完整
4. **投资风险**：本工具仅供参考，不构成投资建议，投资有风险，入市需谨慎

### 1. 即时通知（QQ + 钉钉）

| 通知渠道 | 消息格式 | 触发时机 |

- 💡 操作建议（仓位、策略、选股方向）
- ⚠️ 风险提示

### 2. 盘中实时监测

交易时段自动监测情绪变化：

| 任务 | 时间 | 功能 |
|------|------|------|
| market-emotion-intraday-monitor | 09:30-11:30, 13:00-14:30 每30分钟 | 盘中情绪监测 |

### 3. 转折点预警

自动检测情绪转折点并发送预警：

| 转折点 | 预警内容 |
|--------|---------|
| 冰点期→启动期 | 🌟 机会来临！ |
| 发酵期→高潮期 | ⚡ 情绪加速，注意高位风险！ |
| 高潮期→退潮期 | 🚨 重要转折，注意风险！ |
| 退潮期→冰点期 | ❄️ 情绪恶化，空仓观望！ |

### 4. 数据源统一
