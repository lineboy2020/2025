# 同花顺趋势尾盘策略项目

## 项目目标
基于 `ths-strategy-template` 新建趋势尾盘实战策略项目，纠正旧 `trend_eod_screener` 中“研究版 EOD 候选生成”和“尾盘实时买入筛选”语义混杂的问题。

## 当前项目路径
- `/root/.openclaw/workspace/projects/ths-trend-eod-strategy`

## 当前版本定位（V0）
- 今天尾盘实时筛候选
- 14:50 前买入参考
- 明天按规则卖出
- 暂不接实盘交易执行

## 已完成
- 独立项目目录
- 趋势尾盘专用配置
- 尾盘实时筛选 `runner_live.py`
- 风险标签 / 买入计划骨架
- 历史回测 `runner_backtest.py` 第一版接入
- `market-emotion` 静默适配层接入
- 项目文档落地

## 当前回测语义
- 今日尾盘按候选文件中的 `buy_price_ref` 近似买入
- 次日用 `high/low/close` 近似评估：
  - 先判断 `-7%` 止损
  - 再判断 `+3% / +5% / +8%` 止盈
  - 否则按次日收盘结算

## 当前 V1 配置
- `max_intraday_gain_pct = 7.0`
- `tail_strength_threshold = 0.5`
- `min_turnover_amount = 5e8`
- `max_candidates = 2`

## 当前 V1 市场环境分层
- 发酵期 / 高潮期 → `risk_on` → 做 2 只
- 启动期 → `neutral` → 做 1 只
- 冰点期 / 退潮期 → `risk_off` → 不做
