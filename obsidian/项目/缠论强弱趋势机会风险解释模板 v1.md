# 缠论强弱趋势机会风险解释模板 v1

## 项目定位

本项目的目标不是继续扩展缠论术语，而是把现有缠论算法输出，翻译成可以直接用于看股、复盘、判断交易机会的解释框架。

核心问题：

1. 这只股票当前是 **强 / 中 / 弱**？
2. 当前属于 **上涨趋势 / 中枢震荡 / 下跌趋势 / 弱转强修复 / 强转弱分歧**？
3. 当前有没有 **1买 / 2买 / 3买 / 卖点 / 观望机会**？
4. 结构上最大的 **风险** 在哪里？
5. 能否给出 **合理的涨跌幅度预期区间**？

---

## 输入基础

当前已有缠论算法可提供：

- 分型 `fractal_type`
- 笔 `bi`
- 中枢 `zhongshu`
- 买点 `buy_points`
- 卖点 `sell_points`
- 当前状态 `current_state`
- 背驰（买卖点逻辑中隐含）

项目要做的是在这些结构事实之上，再加一层：

## 解释层（Interpretation Layer）

把结构翻译成：
- 强弱
- 趋势
- 机会
- 风险
- 空间预期

---

## 第一版输出模板目标

### 一、结构概览
- 最后一笔方向
- 最近5笔结构特征
- 当前是否存在中枢
- 当前相对中枢位置：上方 / 内部 / 下方

### 二、强弱评级
- 强
- 中强
- 中性
- 中弱
- 弱

### 三、趋势判断
- 上涨趋势
- 中枢震荡
- 下跌趋势
- 弱转强修复
- 强转弱分歧

### 四、当前机会
- 无机会
- 1买试错
- 2买确认
- 3买跟随
- 中枢突破待回踩
- 高位不追

### 五、当前风险
- 风险低
- 风险中
- 风险高

### 六、涨跌幅度预期
- 向上第一目标位
- 向上延伸目标位
- 关键止损位
- 结构失效位

---

## 第一版设计原则

1. **先解释，不先预测**
   - 先把结构说清楚
   - 再谈机会和空间

2. **先定性，再定量**
   - 先判断强弱与趋势
   - 再给出空间预期

3. **尽量复用现有缠论算法输出**
   - 不急着重写底层算法
   - 先做解释层落地

4. **输出必须说人话**
   - 不是只给 `3B / 2S / zhongshu`
   - 要能直接让人理解当前股票状态
